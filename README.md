# Multimodal Technical Research Assistant with Agentic RAG

An engineering-locked implementation of a multimodal research assistant utilizing LangGraph, FastAPI, pgvector, and Next.js.

## Getting Started

1. Set up your environment variables in `.env` (copy from `.env.example`).
2. Run `docker-compose up --build` to start all services.
3. Access the frontend at `http://localhost:3000` and backend at `http://localhost:8000`.
